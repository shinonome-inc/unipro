プログラミングって難しいの？お金が掛かるの?

皆さんはプログラミングは難しい、

お金がかかると思っていませんか？


たとえプログラミングが重要だったとしても、

難易度がとても高く、
初期費用や
維持費用が
高かったとしたら始めるのは
大変ですよね！

ここでは、プログラミングは

初心者の小学生でも本当にできるのか、


お金が掛からず始めることができるのか
を
お話ししていきます！

プログラミングは難しい？

プログラミングは難しいと思ってる人はとても多い
です。

確かにプログラミングをする際に使うコードと呼ばれる
ものは

一見すると英語の羅列なので特に小学生にはとても
難しく

感じるかもしれません。 しかし、スクラッチというゲーム
を作る
ための
ブロックをパズル感覚で組み立てる
ことに
よって プログラムを組めるので
英語の知識やキーボード
操作
ができなくても全く問題ありません！

スクラッチは
他の一般的なプログラミング言語と呼ばれるもの
に比べて、

簡単に作ることができるのですが、 その一方で、
パズルを
組み立てる際に
論理的思考力なども
しっかりと
鍛えることができます。
したがって、
もし「英語できないん
だけど…」「初心者なんだけど…」
と
プログラミングが
難しいと思っていたとしても、
少しづつ慣れていけば

全く問題ありません！

プログラミングって
お金がかかるの？

続いて多い疑問としてあるのは プログラミングには
初期費用
や
維持費用などの お金がかかるのではないか？

というもの
です。 しかし、先に紹介したスクラッチは

入会費や年会費
などは一切かからず、 誰でも無料
で
使うことができます。
そのため、パソコン１台あれば
手軽に始める
ことができます！
例えば「作ったホームページを
公開したい」みたいな目的が

あった場合、 お金がかかる
場合もありますが、それでも
数百円、
数千円程度ですし、
小学生が自分でゲームを作る
という目的で
あれば
無料でも始めることも十分可能です。
